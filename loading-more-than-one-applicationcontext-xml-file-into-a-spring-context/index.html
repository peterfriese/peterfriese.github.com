Loading More Than One applicationContext XML File Into a Spring Context

Jan 26th, 2005

You can use this command to load more than one application context file into a spring context:

String serviceContextLocation = "service.xml"; String serverContextLocation = "server.xml"; ClassPathXmlApplicationContext combinedContext = new ClassPathXmlApplicationContext(new String[] {serviceContextLocation, serverContextLocation});
